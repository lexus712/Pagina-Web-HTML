Aprende los conceptos fundamentales de Python para empezar a programar. Esta sección cubre lo esencial de forma resumida.

¿Por qué Python?

Python es un lenguaje de programación fácil de aprender y con una sintaxis clara. Es ideal para principiantes y utilizado ampliamente en áreas como desarrollo web, ciencia de datos e inteligencia artificial.

Declaración de Variables

En Python, no necesitas especificar el tipo de dato. Simplemente asigna un valor a una variable.

# Ejemplo de variables
nombre = "Juan"
edad = 25
pi = 3.14159

Python infiere automáticamente el tipo de datos.

Estructuras Condicionales

Usa la sentencia if, elif y else para tomar decisiones en tu programa.

# Ejemplo de condicional
edad = 20
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Bucles

Python tiene dos tipos principales de bucles: for y while.

Bucle for

# Ejemplo de bucle for
for i in range(5):
    print(i)  # Imprime los números del 0 al 4

Bucle while

# Ejemplo de bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Funciones

Las funciones en Python se definen con la palabra clave def. Te permiten reutilizar código.

# Ejemplo de función
def saludar(nombre):
    print(f"Hola, {nombre}")

saludar("Carlos")  # Llama a la función

Comentarios

Usa el símbolo # para escribir comentarios en tu código. Los comentarios ayudan a documentar tu programa.

# Esto es un comentario
print("Hola, Mundo")  # Esto imprime un mensaje
